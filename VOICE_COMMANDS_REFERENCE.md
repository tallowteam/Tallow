# Voice Commands Quick Reference

## How to Use

1. Click the microphone button in the bottom-right corner
2. Wait for the "Listening..." indicator
3. Speak a command clearly
4. The app will execute the command and provide voice feedback

## Available Commands

### File Operations

| Command | Variations | Action |
|---------|-----------|--------|
| Send file | "send file", "upload", "send" | Opens the file selector to send files |
| Receive file | "receive file", "download", "receive" | Switches to receive mode |

### Connection

| Command | Variations | Action |
|---------|-----------|--------|
| Connect | "connect", "connect to device" | Opens device connection dialog |

### Navigation

| Command | Variations | Action |
|---------|-----------|--------|
| Settings | "settings", "show settings", "open settings" | Opens the settings page |
| Help | "help", "what can I say", "commands" | Shows available voice commands |

### Control

| Command | Variations | Action |
|---------|-----------|--------|
| Stop listening | "stop listening", "stop", "deactivate" | Disables voice commands |

## Tips for Best Results

### Clear Speech
- Speak clearly and at a normal pace
- Use a quiet environment when possible
- Speak directly at your microphone

### Natural Language
- Commands are flexible - use variations that feel natural
- You don't need to memorize exact phrases
- The system recognizes partial matches

### Feedback
- Wait for voice confirmation before giving next command
- Watch for the visual transcript to verify recognition
- If misunderstood, try rephrasing

## Privacy

### Your Voice Data
- ✅ Processed locally in your browser
- ✅ Never sent to external servers
- ✅ Not recorded or stored
- ✅ Requires explicit permission

### Microphone Access
- Only active when you click the microphone button
- Automatically stops after each command (non-continuous mode)
- Can be disabled at any time in settings

## Troubleshooting

### Command Not Recognized
1. Speak more clearly or louder
2. Try alternative command phrases
3. Check microphone is working
4. Ensure microphone permission is granted

### No Voice Feedback
1. Check "Voice feedback" is enabled in settings
2. Verify system volume is turned up
3. Test system text-to-speech in browser settings

### Microphone Permission Denied
1. Click the lock icon in the address bar
2. Change microphone setting to "Allow"
3. Reload the page
4. Try again

## Keyboard Shortcuts

While voice commands are active:
- No keyboard shortcuts needed - all voice controlled
- Click microphone button to toggle on/off

## Settings

Access voice command settings:
1. Navigate to Settings page
2. Find "Voice Commands" section
3. Available options:
   - Enable/Disable voice commands
   - Toggle voice feedback
   - View available commands
