\## \*\*20.0 DEEP-DIVE AUDIT REMEDIATION (THE "FIX" LIST)\*\*

\*\*Source:\*\* `TALLOW\_GRAND\_AUDIT\_SYNTHESIS.md`, `WORKER\_ARCHITECTURE\_AUDIT\_REPORT.md`

\*\*Section Summary:\*\* Specific technical debt and critical vulnerabilities identified during the Feb 2026 Grand Audit. These are mandatory engineering tasks.



\### \*\*20.1 Web Worker Architecture Fixes (Agent 028)\*\*

\-   \*\*Critical Security Vulnerabilities:\*\*

&nbsp;   -   ☐ \*\*Missing Worker:\*\* `lib/workers/compression.worker.ts` is missing. \*Action: Implement or remove reference.\*

&nbsp;   -   ☐ \*\*Path Resolution:\*\* `new Worker('../path')` fails in production. \*Action: Use `new URL('./worker', import.meta.url)` pattern.\*

&nbsp;   -   ☐ \*\*Input Validation:\*\* Worker message payloads lack Zod validation. \*Action: Install Zod and validate inputs.\*

&nbsp;   -   ☐ \*\*Memory Leaks:\*\* `postMessage` used without Transferable Objects. \*Action: Refactor to use transfer list for ArrayBuffers.\*

&nbsp;   -   ☐ \*\*Restart Race:\*\* Workers die without draining tasks. \*Action: Implement task re-queuing on worker death.\*

&nbsp;   -   ☐ \*\*SharedArrayBuffer:\*\* Usage without `COOP/COEP` checks causes crashes. \*Action: Add runtime feature detection.\*

\-   \*\*High Severity Issues:\*\*

&nbsp;   -   ☐ \*\*Leakage:\*\* Error messages leak file paths/variable names. \*Action: Sanitize errors in production.\*

&nbsp;   -   ☐ \*\*IP Leak:\*\* `network.worker.ts` exposes local IPs/mDNS. \*Action: Filter ICE candidates.\*

&nbsp;   -   ☐ \*\*Blocking:\*\* `Atomics.wait()` used on main thread. \*Action: Wrap in worker check.\*



\### \*\*20.2 Grand Audit Blockers (Division Chiefs)\*\*

\-   \*\*Crypto (Division Alpha):\*\*

&nbsp;   -   ☐ \*\*Fake PAKE:\*\* CPace/OPAQUE implementations are stubs. \*Action: Replace with `@noble/curves` or `opaque-ke`.\*

&nbsp;   -   ☐ \*\*KDF Logic:\*\* `deriveMessageKey` ignores message number. \*Action: Fix KDF context binding.\*

&nbsp;   -   ☐ \*\*Timing:\*\* `verifyProof` uses `===`. \*Action: Replace with `constantTimeCompare`.\*

\-   \*\*Network (Division Bravo):\*\*

&nbsp;   -   ☐ \*\*Missing TURN:\*\* Relay server is onion-only, not TURN. \*Action: Deploy `coturn` or Cloudflare TURN.\*

&nbsp;   -   ☐ \*\*Phantom Members:\*\* Socket disconnects don't clean up rooms. \*Action: Fix signaling logic.\*

\-   \*\*Frontend (Division Echo):\*\*

&nbsp;   -   ☐ \*\*Build Config:\*\* `ignoreBuildErrors: true` in `next.config.ts`. \*Action: Remove flag, fix TS errors.\*

&nbsp;   -   ☐ \*\*Silent Fallback:\*\* BLAKE3 WASM failure reverts to SHA-256 without warning. \*Action: Hard failure or explicit UI warning.\*

\-   \*\*Ops (Division Hotel):\*\*

&nbsp;   -   ☐ \*\*CI Branch:\*\* CI checks `main` but repo uses `master`. \*Action: Update workflow triggers.\*

&nbsp;   -   ☐ \*\*Tunnel Port:\*\* Cloudflare tunnel points to 3001, app runs on 3000. \*Action: Correct config.\*
