## Character Set Notes

### Dutch (nl.ts)
- Standard Latin alphabet with diacritics where needed
- Friendly, informal tone using "je/jij"
- Examples: "Veiligheid", "Bestanden", "Overdracht"

### Russian (ru.ts)
- Full Cyrillic alphabet (А-Я, а-я, ё)
- Formal register using "Вы" (formal you)
- Examples: "Безопасность" (Safety), "Шифрование" (Encryption), "Передача" (Transfer)
- All characters properly encoded in UTF-8

### Turkish (tr.ts)
- Turkish-specific characters properly implemented
- İ/ı distinction crucial for proper Turkish
- Special characters: ş, ç, ğ, ü, ö, İ
- Examples: "Güvenlik" (Security), "Dosya" (File), "Paylaş" (Share)

### Polish (pl.ts)
- Polish diacritical marks properly encoded
- Special characters: ą, ć, ę, ł, ń, ó, ś, ź, ż
- Examples: "Bezpieczeństwo" (Security), "Udostępnianie" (Sharing), "Połączenie" (Connection)

## Integration with i18n System

All four translation files integrate seamlessly with the existing Tallow i18n infrastructure:

1. **Types:** Match the interface defined in `lib/i18n/types.ts`
2. **Locales:** Registered in SUPPORTED_LOCALES array
3. **Structure:** Export default object with nested categories
4. **Keys:** All 138 keys present and complete
5. **TypeScript:** Full type safety with no `any` types
