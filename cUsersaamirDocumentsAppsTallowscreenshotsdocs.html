Quick Examples

Get started in seconds with these code snippets

Installbash
npm install tallow
# or
yarn add tallow
Quick Sharetypescript
import { Tallow } from 'tallow';

const tallow = new Tallow();
await tallow.shareFile('document.pdf');
Create Roomtypescript
const room = await tallow.createRoom({
  name: 'Team Files',
  maxPeers: 10
});
console.log(room.code); // ABC123

Security Overview

Tallow uses post-quantum cryptography (Kyber-1024, Dilithium) and ChaCha20-Poly1305 for military-grade encryption. All transfers are end-to-end encrypted.

Learn about security

Room System

Create secure rooms for multi-device file transfers. Each room gets a unique 6-character code that expires after use.

Explore rooms

Chat Integration

Built-in encrypted chat with Triple Ratchet protocol. Communicate securely during file transfers with perfect forward secrecy.

View chat docs

Privacy Features

Automatic metadata stripping for images and videos, onion routing for IP protection, and privacy mode to prevent tracking.
