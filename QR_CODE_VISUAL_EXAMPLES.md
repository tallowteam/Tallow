## Color Scheme Variations

### Monochrome (Best for Scanning)
```
Color 1: #000000 (Black)
Color 2: #FFFFFF (White)

Pattern: High contrast, binary
Use: Printing, scanning, accessibility
```

### Accent (Brand Consistency)
```
Base HSL: (210, 100%, ?)
Variations:
- Light: hsl(210, 100%, 65%)  = #5CB3FF
- Medium: hsl(210, 100%, 50%) = #0080FF
- Dark: hsl(210, 100%, 40%)   = #0066CC

Pattern: Blue color variations
Use: Web display, brand alignment
```

### Gradient (Visual Interest)
```
Full spectrum based on hash:
- Red: hsl(0, 80%, 60%)    = #E64D4D
- Orange: hsl(30, 80%, 60%) = #E6894D
- Yellow: hsl(60, 80%, 60%) = #E6E64D
- Green: hsl(120, 80%, 60%) = #4DE64D
- Blue: hsl(240, 80%, 60%)  = #4D4DE6
- Purple: hsl(270, 80%, 60%) = #894DE6

Pattern: Rainbow variations
Use: Decorative, unique identifiers
```
