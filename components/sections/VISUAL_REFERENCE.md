## Design Patterns Used

1. **Gradient Text**: Vercel-style multi-color gradients
2. **Glow Effects**: Blurred colored backgrounds
3. **Glass Morphism**: Backdrop blur on cards
4. **Scroll Animations**: Intersection Observer triggers
5. **Micro-interactions**: Hover scale and glow
6. **Staggered Animations**: Delayed entrance timing
7. **Skeleton Patterns**: Grid pattern overlays
8. **Floating Elements**: Absolute positioned badges
9. **Counter Animations**: Eased number incrementing
10. **Shimmer Effects**: Gradient sweep on buttons

This visual reference provides a complete overview of the structure, layout, and design patterns used in all landing page sections.
