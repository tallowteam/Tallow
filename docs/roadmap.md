# Tallow Roadmap

## v1.0 — MVP
- Send + receive files
- Post-quantum encryption (ML-KEM-1024 + X25519)
- Code phrase authentication
- P2P + single relay
- CLI with progress bars
- Basic Tor integration

## v1.5 — Enhanced
- TUI mode
- Contact management
- Chat alongside transfers
- Resume interrupted transfers
- Multi-file/directory transfers

## v2.0 — Platform Expansion
- WASM browser client
- Mobile (iOS/Android) via FFI
- Enterprise features (SIEM logging, compliance)
- Multi-relay with geographic routing

## v2.5 — Advanced
- Group transfers (rooms)
- Streaming transfers
- Delta/incremental sync
- Plugin system
